# Mobile Banking Customer Experience Dashboard

## Project Overview

This project is an executive-style Power BI dashboard focused on customer experience and digital banking satisfaction analysis.

The report simulates survey responses from 2500 mobile banking users and presents insights related to:
- Net Promoter Score (NPS)
- Customer satisfaction
- Feature experience ratings
- User preferences and banking habits
- Regional comparison across countries

The dashboard was designed with a modern fintech-inspired visual style, focusing on clean layouts, infographic-style visuals, and executive reporting rather than traditional analytical dashboards.

---

# Dashboard Pages

## 1. Executive Overview
High-level customer loyalty and satisfaction overview including:
- Net Promoter Score (NPS)
- Promoters / Passives / Detractors breakdown
- Country comparison
- Customer loyalty segmentation

---

## 2. Customer Preferences
Behavioral insights related to digital banking habits:
- Biometric vs password login
- Instant vs scheduled payments
- In-app support vs call center support
- Notification preferences

Designed using KPI cards, custom icons, and minimalist progress bars.

---

## 3. Customer Experience Ratings
5-point satisfaction analysis across key mobile banking features:
- App Speed
- Login Experience
- Payment Process
- Navigation
- Card Management

The page uses a custom infographic-style layout to visualize how users rate different banking functionalities from Very Bad to Excellent.
