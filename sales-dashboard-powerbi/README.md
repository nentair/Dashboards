# Sales Performance Dashboard

## Overview

A Power BI dashboard designed to analyze sales performance, profitability, customer activity, and product performance.

The report uses a star schema data model, DAX measures, interactive filters, and bookmark-based navigation.

## Key Metrics

- Total Revenue
- Total Profit
- Total Orders
- Unique Customers
- Average Order Value
- Profit Margin %
- Revenue Growth %

## Dashboard Sections

### Executive Overview
- KPI summary
- Revenue trend analysis
- Sales by category
- Top-performing products

### Compare
- Revenue by country
- Profit margin by category

### Ranking
- Top products by revenue
- Top customers by revenue

## Tools & Technologies

- Power BI
- Power Query (M)
- DAX
- Star Schema Data Modeling

## Features

- Interactive filters
- Bookmark navigation
- Custom KPI cards
- Dynamic visualizations
